Display and manage quotes for WooCommerce products.

== Description ==

Adas Quote for WC is a plugin that allows you to display and manage quotes for WooCommerce products. This plugin provides an easy way for customers to request quotes and for administrators to manage these requests.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/adas-quote-for-wc` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Settings->Adas Quote for WC screen to configure the plugin.

== Frequently Asked Questions ==

= How do I display the quote request form? =

You can use the shortcode `[custom_quote_request]` to display the quote request form on any page or post.

= How do I manage the quotes? =

You can manage the quotes from the WordPress admin area under the "Adas Quotes" menu.
